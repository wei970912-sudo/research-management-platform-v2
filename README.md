# 科研管理平台 V2

本目录是基于旧版科研管理工具重新拆分出的前后端分离工程，旧版目录保持不动，新工程单独维护。

## 目录结构

- `frontend/`: Vue 3 + Vite 管理端
- `backend/`: Spring Boot API 服务
- `docs/`: 架构说明、迁移规划、接口边界说明

## 当前状态

- 已完成前后端分离项目骨架搭建
- 已完成统一数据底座服务与跨模块聚合接口
- 已完成核心模块路由与后端 API 占位接口
- 已完成 OpenAI 兼容科研助手接口，默认面向 Ollama 配置
- 已完成总体架构文档和迁移思路文档
- 暂未执行依赖安装

## 核心模块

- 项目中心
- 进度与里程碑
- 会议记录
- 合同与经费
- 文档归档
- 日历提醒
- 总览驾驶舱
- 科研助手

## 建议启动顺序

1. 在 `backend/` 完成数据库接入与真实业务实现
2. 在 `frontend/` 对接真实接口并补齐表单页
3. 编写数据迁移脚本，把旧版 `localStorage` 数据导入后端数据库
4. 将统一数据底座从内存种子数据切换到真实数据库
5. 将科研助手升级为项目资料检索增强模式

## 说明

- 旧版工程目录未被修改
- 新版以“统一数据中心 + 前后端分离 + 模块化扩展”为目标
- AI 助手接口使用 OpenAI 兼容协议，可直接对接 Ollama 本地模型
